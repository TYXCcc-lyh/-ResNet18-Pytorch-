# -ResNet18-Pytorch-
本研究旨在探索利用计算机视觉技术辅助眼科疾病筛查的可行性，以白内障为例，构建一个能自动分类“正常晶状体”与“白内障疑似病变”的智能诊断模型。项目基于公开的Kaggle ODIR数据集，采用迁移学习方法，借助PyTorch深度学习框架，微调了在大型图像数据集上预训练的ResNet模型。实践表明，该方法能有效构建一个初步可用的分类模型。（鉴于技术原因，暂未上传训练好的模型PTH文件和原始数据集）注：数据集来源https://www.kaggle.com/datasets/gunavenkatdoddi/eye-diseases-classification
