# -ResNet18-Pytorch-
本研究旨在探索利用计算机视觉技术辅助眼科疾病筛查的可行性，以白内障为例，构建一个能自动分类“正常晶状体”与“白内障疑似病变”的智能诊断模型。项目基于公开的Kaggle ODIR数据集，采用迁移学习方法，借助PyTorch深度学习框架，微调了在大型图像数据集上预训练的ResNet模型。实践表明，该方法能有效构建一个初步可用的分类模型，其关键价值不仅在于达到一定的分类准确率，更在于通过可视化技术与错误案例分析，深入解读了模型决策的依据与局限性，例如对图像质量、病变早期特征的敏感度问题。本研究为医学影像分析提供了一个从数据准备、模型构建到结果阐释的完整入门实践案例，论证了AI辅助筛查在提升诊断效率方面的潜力
